# Her is Yen !

 網頁介紹：
> 為方便存取常用的資源所建置的網站，內容含部落格撰寫、喜愛的影片及音樂列表。

開發動機：
>練習網頁製作，首次嘗試用PHP Framework Laravel開發網站

開發應用：
>使用Laravel框架，以MVC架構開發，網頁符合RWD響應式設計，使用HTML、CSS、PHP、 Bootstrap等開發，資料皆從資料庫提取，網站功能含登入註冊、後台介面管理。
